# 五子棋游戏

这是一个使用Python和Pygame库开发的简单五子棋游戏。玩家可以与具有不同难度级别的AI对手进行对战。

## 功能特点

- 图形用户界面，使用Pygame库实现
- 玩家对AI模式
- 三种AI难度级别：简单、中等、困难
- 可调整的游戏难度
- 胜利检测
- 重新开始游戏功能

## 安装要求

要运行这个游戏，您需要安装以下软件：

- Python 3.x
- Pygame库

## 配置

游戏的各种参数可以在 `config.json` 文件中进行自定义。您可以调整以下设置：

- 窗口大小
- 棋盘大小和格子尺寸
- 颜色设置
- 默认 AI 难度
- 字体大小和类型

请根据您的喜好编辑 `config.json` 文件来自定义游戏。
